# Barbearia do Ton

Este projeto é uma landing page para a Barbearia do Ton.

## Como fazer o deploy no Vercel

O projeto já está preparado para o Vercel. O arquivo principal foi renomeado para `index.html`, que é o padrão reconhecido automaticamente.

### Opção 1: Via GitHub (Recomendado)
1. Envie este código para um repositório no GitHub.
2. Acesse [vercel.com](https://vercel.com) e faça login.
3. Clique em "Add New..." > "Project".
4. Importe o repositório do GitHub.
5. Clique em "Deploy". Nenhuma configuração extra é necessária.

### Opção 2: Via Vercel CLI
1. Instale a CLI: `npm i -g vercel`
2. No terminal, dentro desta pasta, rode: `vercel`
3. Siga as instruções na tela.

## Tecnologias
- HTML5
- Tailwind CSS (via CDN)
- Javascript (Vanilla)
